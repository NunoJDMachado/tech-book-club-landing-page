# accessibility

- each star should have an alt text or all empty?
  answer: keep star icons decorative (empty alt) when adjacent text already communicates rating/context. if stars are the only rating signal, provide one accessible text alternative for the whole group (for example: "Rated 5 out of 5").

- in picture elements, should only the img have alt text or should the picture element or the source elements also have?
  answer: only the img inside picture gets alt text. picture and source do not support alt.
